En este ejercicio vamos a crear una aplicación web utilizando la librería `Flask`, la cual nos permite desarrollar paginas web de manera mas sencilla en Python, esta pagina contiene un titulo y una lista de número del 1 al 30.

---

Para realizar este ejercicio primero debemos importar la librería `Flask`, la cual sirve para crear aplicaciones web:
```
	from flask import Flask
```
Luego creamos la aplicación Flask:
```
	aplicacion = Flask(__name__)
```
Después definimos la ruta raíz (`/`). Esta ruta es la que se mostrará cuando abramos la web:
```
	@aplicacion.route("/")
	def raiz():
```
Dentro de esta función se creamos una variable llamada `cadena` que contiene el código HTML que se mostrará en la página:
```
	cadena = '''
	<!doctype html>
	<html>
	  <head>
		<title></title>
		<style>
		  h1{color:red;}
		</style>
	  </head>
	  <body>
		<h1>Esto es HTML a tope de power</h1>
	'''
```

Luego utilizamos un bucle `for` para generar los números del 1 al 30 y agregarlos dentro del HTML:
```
	for dia in range(1,31):
		cadena += '<div>'+str(dia)+'</div>'
```
Luego tenemos que cerrar las etiquetas y devolverle la cadena al navegador:
```
	cadena += '''
	  </body>
	</html>
	'''
	return cadena
```
Y por ultimo verificamos que sea el archivo principal y ejecutamos:
```
	if __name__ == "__main__":
		aplicacion.run(debug=True)
```

---

A continuación el código completo:
```
	from flask import Flask							## IMPORTAMOS LA LIBRERIA FLASK ##

	aplicacion = Flask(__name__)					## CREAMOS UNA APLICACIÓN FLASK (WEB) ##

	@aplicacion.route("/")							## ATRAPAMOS LA RUTA RAIZ (/) ##
	def raiz():										## ENDPOINT ##
	  cadena = '''									
		<!doctype html>
		<html>
		  <head>
		    <title></title>
		    <style>
		      h1{color:red;}
		    </style>
		  </head>
		  <body>
		    <h1>Esto es HTML a tope de power</h1>
	  '''
		  
	  for dia in range(1,31):						## CREAMOS UN BUCLE FOR ##					
		cadena += '<div>'+str(dia)+'</div>' 
		     
	  cadena += '''
		  </body>
		</html>
	  '''
	  return cadena									## DEVOLVEMOS LA CADENA COMO HTML EN LA WEB ##
	 
	if __name__ == "__main__":						## SI ESTE ES EL ARCHIVO PRINCIPAL ##
	  aplicacion.run(debug=True)					## EJECUTA LA WEB ##
```
